Виконав Соловій Денис ПІ-21

Було спроектовано та реалізовано бібілотеку класів для системи розмежування доступу та централізованого зберігання документів.

Форма - це приклад використання системи. У ній створено тестових користувачів з різними ролями, групами та правами доступу.
Для переходу на сторінку користувача потрібно ввести його логін.

Логіни користувачів:

Admin	має всі права доступу
+w	може тільки створювати та редагувати(без перегляду :-) ) файли та документи
-r	може тільки видаляти та читати файли та документи
rw	може тільки записувати та читати файли та документи
+-	може тільки видаляти та створювати файли та документи
group	має всі права доступу(приклад використання групи)

Якщо вписати неіснуючий логін, то буде зареєстрований новий користувач без прав доступу.
